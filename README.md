**ErFold-RNAWann** is the demo/showcase interface for **BEPS**, a toolbox for RNA 3D structure prediction and scoring, developed at **IBISC**.

## Features
- **Home** — landing page with dynamic database stats and a quick sequence search
- **Browse** — search, filter, sort, and score RNA sequences from the database
- **Sequence detail** — interactive 3D structure viewer powered by [Mol*](https://molstar.org/) (MolStar)
- **About** — project description, team, and contact information

## Tech stack

- [React Router v7](https://reactrouter.com/) (framework mode)
- TypeScript
- Tailwind CSS
- MolStar (3D molecular viewer)
